# 20-proof/ -- 実証レベルの定義と証拠（日本語版）

## ★の意味

本リポジトリの★評価は**読者が無料版の情報だけでどこまで検証できるか**で定義しています。

| レベル | 定義 | 読者にできること |
|--------|------|----------------|
| ★★★★★ | 設計思想が完全に公開されており、読者が自環境で構築可能 | 30-adoption/try/ で体験 → 10-framework/ で理解 → 自環境で構築 |
| ★★★★ | 設計思想は公開済みだが、定量データは有料版で提供 | 考え方は検証可能。数値は有料版 |
| ★★★ | 概念と一部の証拠画像のみ。詳細設計・定量データは有料版 | 概念の妥当性のみ判断可能 |

### ★5（3項目）: No.1, No.2, No.12
無料版だけで設計思想を理解し、自分のAIに適用できます。

### ★4（8項目）: No.3, 4, 5, 6, 7, 8, 9, 13
設計思想は理解できるが、定量データは有料版で提供。

### ★3（4項目）: No.10, 11, 14, 15
概念は公開しているが、詳細設計と定量データの両方が有料版。

## 無料版 / 有料版の範囲

各成果の範囲分けの詳細は [成果一覧](../achievements/README.md) を参照（英語版テーブル、EN/JA両リンク付き）。

| レベル | 目的 | 含まれるもの |
|--------|------|------------|
| **無料** | 体験＋方法論 | 30-adoption/try/の制御OS＋10-framework/の構造解説＋FM40件概要＋代表画像5-6枚 |
| **Phase1** | 作り方の考え方 | 設計書10ファイル＋FM60件詳細＋図＋サンプル会話＋reason_code表 |
| **Phase2** | 構造観測の方法 | 全ファイル＋全画像＋FM132件全＋実証ログ＋ソース抜粋 |
| **書籍** | SHI理論でのAI統治 | 全7章＋再現手順（具体的な鍵は一切なし） |

Phase1購入者は差額でPhase2にいつでもアップグレード可能。

- ★3の成果は「実証が弱い」のではなく「公開情報での検証可能性が低い」を意味します
