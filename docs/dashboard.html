3-Layer Separation Simulator
See how separating load by type (not volume) affects quality. Drag the sliders to allocate cognitive load.
What this shows
When a single AI carries judgment + monitoring + execution simultaneously, context pressure builds and quality degrades. Moving judgment to Supervisor (0%) and monitoring to Relay (0%) frees the Worker to focus purely on execution.

This is the core principle of three-layer separation: separate load by type, not by volume.
