## 公開レベル別マスキング方針表

| コンテンツ種別 | 無料版（GitHub/X） | Phase1 | Phase2 | 書籍 |
|-------------|-----------------|--------|--------|------|
| **基本方針** | 方法論・考え方のみ | 考え方の手順＋判断基準 | より深い構造観測の方法 | 理論・哲学・実践 |
| **個人情報・機密** | 完全非公開 | 完全非公開 | 完全非公開 | 完全非公開 |
| **再現鍵**（パス・キー・フルコード・詳細設計） | 一切出さない | 一切出さない | 一切出さない | 一切出さない |
| **画像** | 大幅ぼかし or 構造図のみ | ぼかし＋矢印解説 | 構造図・流れ図中心 | 概念図＋解説中心 |
| **ファイル（.md/.pdf）** | 提供せず（リストのみ） | 提供せず。「こう考えて作った」と解説 | 提供せず。ステップバイステップ方法論 | 提供せず。文章で詳述 |
| **コード・コマンド** | 抽象化した擬似コードのみ | 擬似コード＋考え方 | 擬似コード＋構造観測ポイント | コードなし。設計思想の記述 |
| **公開の目的** | 知らせる：「こんなことをやった人がいる」 | 教える：「作り方の考え方」 | 教える：「構造観測の方法」 | 教える：「SHI理論でのAI統治」 |
